# GIAYXIN DESIGN SYSTEM MASTER v2.0 (PRO MAX)

> [!IMPORTANT]
> This document is the Source of Truth for the Giayxin Radical Redesign. All future modifications must adhere to these tokens and principles.

## 1. IDENTITY & ETHOS
- **Tone**: Radical, Technical, Streetwear, Brutalist.
- **Language**: 100% English (Unified for international appeal).
- **Vibe**: High-contrast, fragmented asymmetric layouts.

## 2. COLOR PALETTE
- **Carbon (Background)**: `#050505` (Custom black with grain texture).
- **Acid Green (Primary)**: `#d4ff00` (Main accent, neon feel).
- **Surface (Card)**: `#0b0b0b` (Subtle separation from background).
- **Muted**: `rgba(212, 255, 0, 0.2)` (Low-opacity Acid Green).

## 3. GEOMETRY (THE ZERO RULE)
- **Border Radius**: `0px` (Strictly enforced for all buttons, cards, containers).
- **Borders**: `1px solid rgba(var(--primary), 0.2)` (Technical, thin borders).
- **Spacing**: Fluid asymmetric spacing (e.g., `translate-y-8` offsets).

## 4. TYPOGRAPHY
- **Display**: `Syne` (Bold/Black) - Used for massive headings.
- **Body/UI**: `Outfit` (Medium/Bold) - Used for labels and text.
- **Styles**: 
  - All CAPS for labels and headings.
  - Tracking/Letter Spacing: `0.2em` to `0.5em` for UI labels.
  - Underscores (`_`) allowed only in technical/code-style labels (e.g., `ARCHIVE_01`).

## 5. MOTION & INTERACTION
- **Duration**: `300ms` - `400ms` (Snappy, responsive). NO durations over 500ms.
- **Easing**: `cubic-bezier(0.4, 0, 0.2, 1)`.
- **Accessibility**: `prefers-reduced-motion` media query MUST be respected (removes all non-critical animations).
- **Cursor**: `Default` pointer (Standard UX). Custom cursors are forbidden.

## 6. TEXTURES
- **Grain Overlay**: Global overlay with `pointer-events-none` for a tactile, raw feel.
- **Scanlines**: Hover effects using linear-gradient animations.
